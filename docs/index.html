MCPWizard

A powerful CLI tool to help you create and deploy Model Context Protocol (MCP) servers with ease.

View on GitHub

Installation

npm install -g mcpwizard

Start building your MCP server in minutes, not hours.

Features

Initialize Projects

Quickly bootstrap new MCP server projects with different templates (TypeScript, Python).

Manage Tools

Add, configure and manage tools for your MCP server with simple commands.

Generate Configs

Automatically generate Claude Desktop configuration files for seamless integration.

Build & Deploy

Build your MCP server for deployment with a single command.

Inspect Servers

Analyze and inspect your MCP servers to ensure they're working correctly.

Active Development

MCPWizard is actively maintained and regularly updated with new features.

Quick Start

# Initialize a new MCP server project
mcpwizard init my-mcp-server

# Navigate to the project directory
cd my-mcp-server

# Add a tool to your server
mcpwizard tool add my-tool --description "My awesome tool"

# Generate Claude Desktop configuration
mcpwizard generate-claude-config
