MCPWizard

A powerful CLI tool to help you create and deploy Model Context Protocol (MCP) servers for Claude and other AI models.

View on GitHub

Installation

npm install -g mcpwizard

Start building your MCP server in minutes, not hours.

Quick Start

# Initialize a new MCP server project
mcpwizard init my-mcp-server --template typescript

# Navigate to the project directory
cd my-mcp-server

# Add a tool to your server
mcpwizard tool add my-calculator --description "A calculator tool for basic math operations"

# Generate Claude Desktop configuration
mcpwizard generate-claude-config

# Inspect your MCP server
mcpwizard inspect
